# FoodWagon Clone from figma design

A modern food ordering frontend built with Next.js, typescript and Tailwind CSS.

## Features

- Responsive design
- Fake API data fetching
- Loading skeletons
- Shopping cart with Context API
- Cart drawer/sidebar
- Reusable UI components
- Mobile responsive layout

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React Context API
- Lucide Icons
